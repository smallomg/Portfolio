## Projects Timeline

### 1. BNK Financial Platform (Web & Mobile Integration)
> **BNK 금융 서비스를 벤치마킹한 차세대 금융 플랫폼 구축 프로젝트**  
> **기간:** 2025.06 ~ 2025.08 (팀 프로젝트)

금융 서비스의 **운영(Back-office)부터 데이터 분석(BI)까지 전반적인 관리자 사이클**을 전담했습니다.  
데이터 무결성을 위한 **승인 결재 시스템**과 사용자 행동 로그 기반의 **개인화 추천/분석 대시보드** 구축에 주력했습니다.

- **Tech Stack**: `Java`, `Spring`, `OracleDB`, `MyBatis`, `Chart.js`, `Apache POI`
- **Key Features**: 상품 승인 결재(Maker-Checker), 권한 관리(RBAC), 개인화 추천 알고리즘, BI 대시보드

**[상세 구현 내용 & 소스코드 보러가기](https://github.com/smallomg/Portfolio/tree/main/BNK-Project)**
